⚙️ Firebase Setup Needed — 2 Minutes, FREE

Your screenshot shows Connecting... because Firebase config is still REPLACE_ME or has a syntax error.

How to fix (do this in your local file, then push again):
  1. Open index.html you pushed in Notepad. Find const firebaseConfig = {
  2. Replace ENTIRE object with your real config from Firebase Console > Project Settings > General > Your apps > Web
  3. It must look like:
    const firebaseConfig = { apiKey: "AIzaSyA...", authDomain: "work-checklist-xxxxx.firebaseapp.com", projectId: "work-checklist-xxxxx", storageBucket: "work-checklist-xxxxx.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123" };
  4. Make sure there are commas, quotes correct. No trailing comments inside.
  5. Save, then in GitHub Desktop or command: git add index.html → git commit -m "fix firebase config" → git push
  6. Wait 2 min for Pages to rebuild (check Actions tab in GitHub - green tick).
  7. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (iPhone - close tab reopen).

If you DID replace, common mistakes: ❌ Forgot to push after editing local file (you edited locally but GitHub still has old REPLACE_ME) ❌ Enabled Authentication?
