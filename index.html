⚙️ Firebase Setup Needed — 2 Minutes, FREE

This sync version needs your FREE Firebase project to store data securely. Data is NOT stored in GitHub — only in your private Firebase.

  1. Go to console.firebase.google.com → Add Project → Name: work-checklist → Disable Google Analytics (optional) → Create
  2. After created: Left menu → Build > Authentication → Get Started → Email/Password → Enable → Save. Also enable Google provider if you want one-click login.
  3. Left menu → Build > Firestore Database → Create Database → Start in Test mode (or Production) → Location: asia-southeast1 (Singapore) for MY → Enable
  4. Top right → Project Settings (gear icon) → General tab → Your apps → Web app </> → Register app: work-checklist → Copy the firebaseConfig object
  5. Paste that config below in this file: open index.html in Notepad/VS Code, find const firebaseConfig = { and replace the placeholder values. Then git commit + push again.
  6. Optional but secure: In Firestore → Rules tab, paste rules from firestore.rules file in this repo, then Publish.
